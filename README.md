# 🎮 Steam Clone — Django

Полный клон Steam на Python Django с тёмной темой, точь-в-точь как оригинал.

## 📦 Установка

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Применить миграции
python manage.py migrate

# 3. Создать суперпользователя (администратора)
python manage.py createsuperuser

# 4. Запустить сервер
python manage.py runserver
```

Открыть в браузере: **http://127.0.0.1:8000**

## 🔧 Панель администратора

http://127.0.0.1:8000/admin/

В панели можно добавлять:
- **Игры** (название, описание, цена, скидка, изображения)
- **Категории** (Экшн, RPG, Стратегия и т.д.)
- **Теги** (Мультиплеер, Одиночная игра и т.д.)
- **Скриншоты** к играм

## ✅ Что включено

- 🏠 Главная страница с каруселью featured-игр
- 🛍️ Магазин с поиском и фильтрами (цена, жанр, сортировка)
- 🎮 Страница игры с галереей скриншотов, описанием, отзывами
- 🛒 Корзина + Оформление покупки
- 📚 Библиотека купленных игр
- ❤️ Список желаемого
- 👤 Регистрация, вход, профиль пользователя
- ⭐ Система отзывов (рекомендую / не рекомендую)
- 🎨 Точная копия дизайна Steam (тёмная тема)
